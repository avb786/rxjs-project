Change Detection

So basically whenever we update some data in ts variables and show on the template , and when it gets manipulate with UI(template) then the changes happen is basically know as Change Detection. In simple words , it synchronises the model changes to view.

Should Change Detection happen?
As soon as an application state changes.
What can cause state change?
  • Event callback (e.g click, change)
  • Network call (XHR)
  • Timer (setTimeout, setInterval)
-So a common factor or similarities of these are , they are asynchronous and kept to change the state
